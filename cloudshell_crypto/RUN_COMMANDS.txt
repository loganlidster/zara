# Crypto Event Generation - Run Commands

## Step 1: Upload and Deploy (ONE TIME)
cd ~/cloudshell_crypto
chmod +x deploy.sh
./deploy.sh

## Step 2: Run EQUAL_MEAN (First Run)
gcloud run jobs execute crypto-event-job --region us-central1 --update-env-vars METHOD=EQUAL_MEAN,BUY_MAX=3.0,SELL_MIN=0.1,SELL_MAX=3.0,START_DATE=2024-05-01,END_DATE=2025-11-02

## Step 3: Run WINSORIZED (Second Run - after EQUAL_MEAN completes)
gcloud run jobs execute crypto-event-job --region us-central1 --update-env-vars METHOD=WINSORIZED,BUY_MAX=3.0,SELL_MIN=0.1,SELL_MAX=3.0,START_DATE=2024-05-01,END_DATE=2025-11-02

## Monitor Progress
https://console.cloud.google.com/run/jobs?project=tradiac-testing

## Check Results After Completion
# EQUAL_MEAN events
SELECT COUNT(*) FROM trade_events_crypto_equal_mean;

# WINSORIZED events
SELECT COUNT(*) FROM trade_events_crypto_winsorized;
